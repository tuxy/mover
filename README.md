# Mover ROS2 Testbed

A small project to convert an old iRobot Essential into a ROS2 Testbed, with simple and cost-effective LiDAR-based navigation. Might rename the project to the twenty-dollar project (since I paid 20 dollars for the robot).

 - `mover_controller`: closed-loop motor controller with a Pi Pico and the L298N (WIP)
 - `mover_pcb`: organised pcb for the pi pico, motors and motor-controller to interface nicely
 - `ros_ws`: workspace containing the packages & description for the robot.
